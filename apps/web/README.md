# syrnike13 Web (React)

React-фронтенд мессенджера syrnike13.

## Стек

- [TanStack Start](https://tanstack.com/start) + Router + Query + Form
- [React 19](https://react.dev/)
- [shadcn/ui](https://ui.shadcn.com/) (CLI)
- [Tailwind CSS v4](https://tailwindcss.com/)
- Типы API: `@syrnike13/api-types` из `packages/api-types`

## Бэкенд

По умолчанию — продакшен **syrnike13.ru**:

| Переменная | Значение |
|------------|----------|
| `VITE_API_URL` | `https://syrnike13.ru/api` |
| `VITE_WS_URL` | `wss://syrnike13.ru/ws` |
| `VITE_MEDIA_URL` | `https://syrnike13.ru/autumn` |
| `VITE_PROXY_URL` | `https://syrnike13.ru/january` |

Скопируйте `.env.example` → `.env` при необходимости.

## Запуск

Из корня репозитория:

```bash
pnpm install
pnpm --filter @syrnike13/web dev
# или: mise dev:web
```

Откройте http://localhost:3000

## shadcn

```bash
cd apps/web
pnpm dlx shadcn@latest add card
```

## Что уже есть

- Вход по email/паролю (`/login`) → продакшен API
- MFA (пароль) при включённой 2FA
- Защищённый `/app` с трёхколоночным layout (серверы · каналы · чат)
- Синхронизация **Ready** и live-событий по WebSocket (сообщения, каналы, серверы)
- Просмотр истории и отправка текста в канале (`/app/c/$channelId`)
- Непрочитанные (бейджи), «печатает…», авто-прокрутка ленты
- Базовый markdown в сообщениях и превью картинок (autumn)
- Вложения, реакции, поиск по каналу, упоминания `@`
- Ответ на сообщение, редактирование и удаление своих
- Подгрузка истории («Загрузить ранние сообщения»)
- Участники сервера для `@` (REST + `ServerMemberJoin` по WS)
- `/app/friends` — друзья, заявки, блокировка, онлайн-статус
- `/app/discover` — встроенный Discover (stt.gg)
- Онлайн-статус по WS `UserPresence` (точка у аватара, подпись в ЛС)
- `/login/register`, `/login/reset`, `/login/reset/$token` — регистрация и сброс пароля
- `/app/settings` — редактирование отображаемого имени
- Подгрузка истории при прокрутке вверх
- **Голос (LiveKit):** подключение к голосовым каналам, синхронизация участников по WS, превью в сайдбаре, панель звонка, микрофон/оглушение/камера/демонстрация экрана, видео на плитках, фон плитки по аватару, индикатор речи, настройки устройств и громкости
- **Серверы:** создание сервера (+), приглашения по ссылке
- **`/invite/$code`** — принять приглашение на сервер
- **Уведомления:** desktop при новых сообщениях; push через `public/sw.js` + VAPID
- **Закрепление:** pin/unpin в меню сообщения, список закреплённых в шапке канала
- **Участники:** боковая панель участников сервера (на широком экране)
- Копирование текста и ID сообщения из меню
- Тема shadcn (светлая/тёмная), Noto Sans JP
- **Сервер:** создание текстового/голосового канала, выход из сервера
- **Переход к сообщению** из поиска и закреплённых (`?m=`)
- Markdown: *курсив*, ~~зачёркнутое~~, цитаты `>`, кастомные emoji `:id:`
- PWA manifest (`/manifest.json`)
- Emoji в composer (unicode + серверные), лайтбокс для картинок
- Профиль пользователя (клик по аватару), копирование ссылки на сообщение
- Роли участников в сайдбаре, «отметить сервер прочитанным»
- Переименование/удаление канала и сервера; переход к цитируемому сообщению
- Виртуализация ленты (60+ сообщений); markdown-спойлеры `||текст||`
- Drag-and-drop и вставка файлов в composer; `@everyone` / `@online`
- Описание канала и сервера; отображение описания в шапке канала
- Сессия в `localStorage`, профиль через `GET /users/@me`
- **ЛС и группы:** кнопка «+» у личных сообщений — новое ЛС или группа (`POST /channels/create`)
- **Друзья:** поиск по списку на главной
- **Настройки → Устройства:** список сессий, завершение чужих сессий
- **Настройки → Аккаунт:** смена пароля
- **Сервер:** вкладки emoji (загрузка/удаление) и просмотр ролей; ПКМ по каналу (прочитано, ссылка, инвайт, настройки)
- **Модерация:** kick/ban участника из профиля (сайдбар участников)

## Ещё не перенесено

- Категории каналов, редактирование ролей, баны (список), вебхуки, права канала
- Кастомный статус, черновики
- Контекстное меню ПКМ по сообщениям, MFA в настройках
- Боты, подписки, расширенный профиль (баннер, статус)

## Структура

```
src/
  features/auth/     # API входа, контекст, форма
  features/sync/     # store + обработка WS-событий
  features/events/   # WebSocket gateway (v1)
  features/api/      # REST: сообщения
  components/layout/ # ServerRail, ChannelSidebar, AppShell
  components/chat/   # лента, composer
  routes/app/c/      # страница канала
```
